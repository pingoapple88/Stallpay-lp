1. LINE 官方帳號開立步驟

要讓顧客能收到取餐通知，您需要擁有自己的 LINE 官方帳號。

  1. 前往 LINE Official Account Manager：開啟瀏覽器前往 LINE 官方帳號管理頁面。
  2. 登入或註冊：使用您的個人 LINE 帳號登入，或建立新的商業帳號。
  3. 建立官方帳號：點選「建立官方帳號」，填寫您的攤位名稱、業種等基本資料。
  4. 取得 Channel ID 與 Channel Secret：
  5. 在管理頁面左側選單，找到「設定」>「Messaging API」。
  6. 點選「啟用 Messaging API」。
  7. 建立一個 Provider（例如您的攤位名稱）。
  8. 啟用後，您會看到 Channel ID 和 Channel Secret，請將這兩個數值複製並填入 StallPay 後台的「LINE 設定」中。
  9. 發行 Channel Access Token：
  10. 在同一個 Messaging API 設定頁面，找到「Channel access token (long-lived)」。
  11. 點選「發行（Issue）」，將產生的 Token 複製並填入 StallPay 後台。

2. ECPay 綠界科技金流申請流程

若您選用「進階版」或「旗艦版」，可開啟行動支付與信用卡收款功能。

  1. 註冊綠界會員：前往 綠界科技官網，點選「免費註冊」，建立企業或個人商戶帳號。
  2. 完成身分驗證：依據綠界指示，上傳身分證、存摺封面（個人商戶）或公司登記證明（企業商戶），完成實名認證。
  3. 取得 API 串接金鑰：
  4. 登入綠界廠商後台。
  5. 進入「系統開發管理」>「系統介接設定」。
  6. 找到您的 MerchantID、HashKey、HashIV，將這三項資料填入 StallPay 後台的「金流設定」中。
  7. Apple Pay 開通（選用）：
  8. 在綠界後台的「信用卡收單」設定中，申請開通 Apple Pay 功能。
  9. 需上傳相關審核資料，審核通過後即可在 StallPay 啟用。
  10. 電子發票申請（選用，需企業商戶）：
  11. 在綠界後台申請「電子發票加值中心」服務。
  12. 取得電子發票的專屬 HashKey 和 HashIV，填入 StallPay 後台的「發票設定」中。

4. 刷卡授權與 7 天試用說明

StallPay 提供所有新攤主 7 天免費試用期，讓您能完整體驗系統功能。

  • 授權流程：在您選擇方案並填寫信用卡資料後，系統會進行「新台幣 1 元」的授權測試（不會實際請款），以確認信用卡有效性。
  • 試用期計算：授權成功當日為試用第 1 天。
  • 正式扣款：在試用期滿的 第 8 天，系統才會自動扣除您所選方案的月費。
  • 取消訂閱：若您在 7 天試用期內決定不繼續使用，可隨時在後台取消訂閱，不會產生任何費用。

5. 各方案設定差異

根據您選擇的方案，StallPay 後台的設定項目會有所不同：

  • 入門版（現金版）：
  • 專注於現金收款，無需設定 ECPay 金流。
  • 無電子發票設定區塊。
  • 適合單一攤位、免開統一發票的小型攤商。
  • 進階版：
  • 需完成 ECPay 金流設定，以開啟 LINE Pay、信用卡、Apple Pay 收款。
  • 可設定電子發票（需為企業商戶）。
  • 可在後台新增最多 3 個攤位，並指派最多 10 個員工帳號。
  • 旗艦版：
  • 包含進階版所有設定。
  • 支援 10 個以上攤位及 50 個以上員工帳號的大型管理。
  • 享有優先客服通道，並可預約專人到府輔導設定。

若在設定過程中有任何疑問，請聯繫您的經銷夥伴，他們將提供在地真人輔導服務！
